LSP Salt Password
=================

Плагин для [LiveStreet CMS](http://livestreetcms.com/ "LiveStreet CMS")
-----------------------------------------------------------------------


Внимание
--------
Плагин требует **LiveStreet CMS версии 2.0 или более поздней**.


Описание
--------
Плагин заменяет стандартную функцию шифрование пароля на более надежную.
